# Angular Starter Kit

Boilerplate para aplicaciones con Angular 1.x utilizando componentes.

## Instalación
```shell
  $ git clone https://github.com/jorcalle11/angular-starter-kit.git
  $ cd angular-starter-kit
  $ npm install
```
## Ejecución
```shell
  $ npm start
```
